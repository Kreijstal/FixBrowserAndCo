This library allows you to interface with native input/output facilities.

Features:

  • currently supported systems: Windows, Linux, Mac OS X, Haiku, WebAssembly
  • both synchronous and asynchronous API (callback based)
  • GZIP compression/decompression
  • TCP/IP network connections
  • access to SQLite and PostgreSQL databases
  • console based user interfaces
  • advanced prompt implementation with history, autocompletion and searching
  • allows integration with foreign event loop
  • licensed under ZLIB license (no attribution required in binary builds)

Synchronous I/O classes

  • Path - path to file or directory
  • Stream - synchronous stream
    • ArrayStream - stream backed by arrays
    • BufferedStream - adds buffering to another stream
    • ZStream - inflate/deflate compression stream
      • GZipStream - GZip compression stream
    • File - opened file
    • TCPConnection - TCP connection
    • ProcessStream - process stream
  • TCPServer - TCP server
  • WebSocket - WebSocket
  • Process - process
  • ZipReader - ZIP reader

Asynchronous I/O classes

  • AsyncStream - asynchronous stream
    • AsyncTCPConnection - asynchronous TCP connection
  • AsyncTCPServer - asynchronous TCP server

Atomic I/O classes

  • AtomicFile - allows atomic updates of files using transactions
  • transaction - transaction statement allows easy usage of transactions

SQL classes

  • SQLConnection - connection to a SQL database
    • SQLiteDatabase - SQLite database
    • PostgreSQLConnection - connection to PostgreSQL server
  • ResultSet - access to query results
    • SQLiteResultSet - SQLite result set
  • ParamString - parametrized SQL string
  • SQL token processor - SQL token processor allows easy usage of SQL parameters

Console user interface

  • Console - advanced access to the console
  • ConsoleEvent - console event
  • PromptHandler - provides dynamic functionality in console prompts
  • ScreenBuffer - off-screen text buffer with attributes

Time classes

  • Clock - access to the current time
  • Monotonic - access to the monotonic time

Global functions

  • Console input/output
  • Asynchronous I/O
  • Compression
  • Checksum
  • Shared arrays
  • Serialization
  • Charset conversion
