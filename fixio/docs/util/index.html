This library contains various utility functions and classes.

Features:

  • basic modules written in pure FixScript (with class definitions in attributes) for better reuse
  • some operations are optionally accelerated using a native code
  • licensed under ZLIB license (no attribution required in binary builds)

Modules

  • String processing
  • General sorting support
  • Base64 encoding & decoding
  • JSON encoding & decoding
  • Long - 64-bit signed integer
  • Double - 64-bit float
  • BigInteger - arbitrary big integer
  • Unsigned arithmetic
  • Struct allocator
