import "util/bigint";

BigInteger class

This class allows to work easilly with arbitrary big integers.

Supported operators: +, -, *, /, %, <<, >>, <, >, <=, >= with BigInteger and Integer types.
A short variant of the constructors is available: BigInteger(...)

Note: the non-class functions are using a shorter bigint prefix for functions.
