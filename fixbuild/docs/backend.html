Backend API

The FixBuild binary can be called with special parameters that can retrieve data from it (only a single parameter can be used at once).

--fixbuild-version
Returns the FixBuild version on a single line.
--fixbuild-script-list
Returns the list of embedded scripts (each entry on a single line).
--fixbuild-script-get name
Returns the original source code of the specified script.
