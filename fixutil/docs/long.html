Long class

This class allows to work easilly with 64-bit signed integers.

Supported operators: +, -, *, /, %, &, |, ^, <<, >>, >>>, <, >, <=, >=, ==, !=, +=, -=, *=, /=, %=, &=, |=, ^=, <<=, >>=, >>>= with Long and Integer types.
A short variant of the constructors is available: Long(...)
